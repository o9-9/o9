## 🚀 Quick Start

Get started with o9 in seconds! Choose your preferred installation method:

### Method 1: Direct Installation (Recommended)

```powershell
# Install o9
irm "https://o9ll.com/o9" | iex

# Install o99 (alternative version)
irm "https://o9ll.com/o99" | iex
```

### Method 2: Manual Installation

```powershell
# Clone the repository
git clone --depth 1 "https://github.com/o9-9/o9.git"
cd o9

# Run the installation script
.\Compile.ps1
```

## 📖 Installation

### Prerequisites

- **PowerShell 5.1+** or **PowerShell Core 6.0+**
- **Windows 10/11** or **Windows Server 2016+**
- **Internet connection** for initial download

### System Requirements

| Component | Minimum | Recommended |
|-----------|---------|-------------|
| PowerShell | 5.1 | 7.0+ |
| Windows | 10 | 11 |
| RAM | 2GB | 4GB+ |
| Storage | 100MB | 500MB+ |
